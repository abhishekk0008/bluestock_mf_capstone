# 📊 Mutual Fund Industry Analytics

A comprehensive end-to-end data analytics project focused on analyzing the Indian mutual fund industry using **Python, SQL, and Power BI**. The project transforms raw financial datasets into meaningful insights through data cleaning, feature engineering, advanced financial metrics, and interactive dashboards.

---

## 🚀 Project Overview

The objective of this project is to analyze mutual fund performance, investor behaviour, SIP trends, and benchmark indices to support data-driven investment analysis.

The project follows the complete analytics workflow:

* Data Collection
* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Advanced Financial Analytics
* Interactive Power BI Dashboards
* Business Insights & Reporting

## 📊 Financial Metrics

The project includes calculation of:

* Daily Return
* CAGR
* Sharpe Ratio
* Sortino Ratio
* Alpha
* Beta
* Maximum Drawdown

---

## 💡 Key Insights

* Identified leading fund houses based on Assets Under Management (AUM).
* Analyzed monthly SIP inflow trends across the industry.
* Evaluated mutual funds using risk-adjusted performance metrics.
* Compared fund performance with benchmark market indices.
* Explored investor behaviour and category-wise investment trends.

## ▶️ How to Run

1. Clone the repository.
2. Install the required Python packages.
3. Run the notebooks in sequence.
4. Open the Power BI dashboard (.pbix) file.
5. Explore the interactive dashboards.
